Core Concept: This engine uses a "Target-Seek" pattern (Linear Interpolation or LERP) for everything. We never instantly change a value; we set a Target, and the render loop smoothly moves the current value toward that target every frame.

The 4-Step Workflow of a State Change:

The Trigger (setMode):

When an event happens (e.g., setMode('CODING')), we do not change the shape instantly.

Instead, we update the Targets: t.targetPoints = 4, t.targetColor = '#00ff00'.

We also set this.state.mutation = 1.0. This is a "chaos variable" that decays over time.

The Mutation Phase (High Chaos):

The draw() loop applies Chromatic Aberration (RGB Split) and Screen Shake based on the mutation value.

The agents scatter randomly (mutationOffsetX) to simulate breaking apart before reforming.

The Morphing Phase (Smooth Transition):

The Settle (Idle Return):

To return to normal, we simply call setMode('IDLE').

This sets the targets back to the default Star properties.

The LERP logic automatically handles the smooth reverse transformation.
